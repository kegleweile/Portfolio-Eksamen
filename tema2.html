TEMA 2 - Grundlæggende Web

Tema og projekt beskrivelse:

I tema 2 lærte vi om grundlæggende forståelse af HTML og CSS struktur. Vi blev introduceret til visuelt design, typografi, html validering, og responsivt webdesign som dækker over grid, flexbox og media queries. Ift. design, der fik vi praktiske færdigheder i grafik, billedbehandling og designprincipper såsom de 5 gestaltlove, wireframes og farve kontraster.

Vores projekt i tema 2 var at lave et responsivt mobile og desktop site omkring computere. Vi fik udleveret layout diagrammer, wireframes og billedemateriale, som vi skulle bruge og selv indsætte det på sitet. Dette var også vores studiestartsprøve. I dette projekt gjorde jeg brug af den viden, som vi havde fået ift. farvekontrast, grid samt typografi (download af fonts, font-face).
